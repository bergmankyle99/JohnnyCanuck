All pages use Nav Styles which use e > f and e f (pc_home_styles.css > NAV STYLES)

2.4.2.5 - Use of CSS3 background styles:

2.4.2.7 - Define document content with the class and id properties and create styles for them:

Contact Us and all pages use classes and id's to style, but one example is the Navigation styles using classes (jc_home_styles.css > NAV STYLES), and the form uses id's for styling (jc_contactUs.css > FORM STYLES)

2.4.2.8 - Use cascading styles to control page layout by using either grid, flex, float, positioning:

Podcasts uses a grid layout(jc_grids.css/jc_podcasts.css)

2.4.2.9 - Use of CSS3 text and box shadows:

Contact Us uses box-shadow around the form (jc_contactUs.css > FORM STYLES)

2.4.2.10 - Use of CSS3 transform rotate feature:

All pages have a transform feature on them when hovering over CTA's at top of screen (jc_home_styles.css > TOP PAGE STYLES)

2.4.2.11 - Use of CSS3 opacity applied to two images to appear semi-transparent:

About has an opaque image in bio (jc_about.css > BIO STYLES),
